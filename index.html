HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Section elements are semantic elements
  • Tags have an opening and a closing
  • Closing Tags have a forward slash before the name

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors can be assigned by semantic references or hex codes
  • Stylesheet can be inline, internal or external
  • Class selector is used to apply style to specific elements

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all newly modified files to version control
  • git push: pushes any commited files to github ready to be merged
  • git pull: brings all changes that has been pushed from github to local workspace

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An Array stores multiple data within it
  • A function holds a block of reusable code
  • A function needs to be called before it can execute
  • for loop checks a block of code to see if it's truthy or falsy
  • Array index starts from 0 and increments by 1
